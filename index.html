Browser benchmarks - February 2025

Test Methodology

  • Chrome V.132.0.6834.163
  • Firefox V.135
  • Tests are run in two separate devices.
  • A Xiaomi Mi 11 Lite 5G that is almost exclusively used for these tests.
  • A Samsung S23 that is my daily driver phone.
  • Data is cleared on both browsers before running the tests
  • Background apps are cleared.
  • At least 2 minutes are allowed between each of the tests so thermal throttling isn't an issue.
  • Firefox tracking protection is disabled.
